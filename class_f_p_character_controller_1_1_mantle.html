Detailed Description

This class handles the entire mantle movement where the player is able to climb onto certain objects depending on their height and other factors.

Member Function Documentation

◆ BeginMantle()

void FPCharacterController.Mantle.BeginMantle ( )
inlineprivate

Initiates the mantling action for the player. This method handles the key inputs used to mantle and the middleOfPlayerRay also prevents any unintended or abnormal movement whilst mantling such as diagonal movement and once the player has reached a certain point in the mantle movement, both the lerpRay and overlapBox are used to end the mantle movement and begin the lerp action which creates a smooth transition from the current mantle position to the top of the object that is being mantled.
