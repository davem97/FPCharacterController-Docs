Detailed Description

This class handles the ladder functionality.

Note: Adjusting the scale of a ladder might require you to manually alter the values of the transforms and trigger areas to match the new ladder scale. To allow for easy ladder entering, the dotProduct and distance inside the two respective bools can be adjusted, as well as the size of the Cube trigger.
